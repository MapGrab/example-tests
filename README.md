# MapgrabDemo tests (MacOS ARM only)

### 1. - Setup .npmrc with private access token

```bash
  npx -y @mapgrab/npm-setup@latest 123456
```

---

### 2. - Install deps

```bash
  npm install
```

### 3. - Run tests

```bash
  npm run test
```
